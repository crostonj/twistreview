# twistreview

A Java-based REST Web API using Spring Boot.

## Features
- Spring Boot REST API
- Sample controller with health check endpoint

## Getting Started

### Prerequisites
- Java 17 or later
- Maven 3.6+

### Build and Run

```
mvn spring-boot:run
```

The API will be available at `http://localhost:8080`.

### Health Check

```
GET http://localhost:8080/health
```
